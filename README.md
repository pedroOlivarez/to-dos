# Pedro Olivarez

## Setup

1. Install dotnet sdk

- https://dotnet.microsoft.com/en-us/download/dotnet/10.0

2. In Api directory

- create a copy of appsettings.json and name it appsettings.Development.json
- request an invite to neon.tech workspace to get access to connectionstring and paste value under "Neon": {"ConnectionString"}

3. Install latest version of Node

- https://nodejs.org/en/download

## API

1. Open terminal
2. Run: `npm run dev:api`

## Client

1. Open another terminal
2. Run: `npm run dev:ui`
